📖 Best practices

  • When to use XML tags: separate instructions, context, examples, constraints — improves Claude’s adherence.
  • Naming: lowercase, hyphenated (e.g. output_format). Keep consistent.
  • Few-shot: use <example> tags with input/output pairs.
  • Role-play: define persona with <persona> tag.
  • Nesting: use indent buttons to create hierarchy (simulated).
