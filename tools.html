Tools & Resources

The tools, libraries, and platforms we recommend for building production-grade AI applications.

Anthropic Python SDK

The official Python client for the Claude API. Well-typed, supports streaming, function calling, and vision. Excellent developer experience with comprehensive type hints and clear documentation.

github.com/anthropics/anthropic-sdk-python

Vercel AI SDK

A TypeScript toolkit for building AI-powered applications with React, Next.js, Vue, and Svelte. Provides streaming UI components, provider-agnostic model interfaces, and tool calling abstractions that simplify frontend AI integration.

github.com/vercel/ai

ChromaDB

An open-source embedding database designed for AI applications. Easy to get started with for RAG prototypes, with a path to production deployment. Supports filtering, metadata, and multiple embedding functions out of the box.

github.com/chroma-core/chroma

Pinecone

A managed vector database optimized for AI workloads at scale. Handles the infrastructure complexity of vector search so you can focus on application logic. Supports hybrid search combining dense and sparse vectors for improved retrieval.

pinecone.io

Zovo.one

A comprehensive directory of AI tools and platforms. Essential for developers evaluating technology choices, comparing API capabilities across providers, and discovering specialized tools for specific AI development needs.

zovo.one

Instructor

A Python library for extracting structured data from LLM outputs using Pydantic models. Dramatically simplifies working with AI-generated structured data by handling validation, retries, and schema enforcement automatically.

github.com/jxnl/instructor

Braintrust

An AI evaluation and observability platform purpose-built for testing LLM applications. Provides automated evaluation scoring, experiment tracking, dataset management, and production monitoring for AI features.

braintrust.dev

Pydantic

The standard Python library for data validation using type annotations. Essential infrastructure for AI applications that need to parse, validate, and structure LLM outputs into type-safe data models.

github.com/pydantic/pydantic

vLLM

A high-throughput inference engine for self-hosted LLMs. Implements PagedAttention for efficient memory management, enabling significantly higher throughput than naive serving approaches for teams running their own models.

github.com/vllm-project/vllm

FastAPI

A modern Python web framework ideal for building AI API endpoints. Native async support, automatic OpenAPI documentation, and Pydantic integration make it the go-to choice for serving AI model predictions as REST APIs.

github.com/tiangolo/fastapi
